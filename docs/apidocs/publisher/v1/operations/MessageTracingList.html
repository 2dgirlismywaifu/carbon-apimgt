Message Tracing List

Get the custom sequence message trace list

GET https://apis.wso2.com/api/am/publisher/v1/apis/{apiId}/message-traces

This operation can be used to retrieve custom sequence message trace list of an API.

OAuth 2.0 Scope

apim:api_view
